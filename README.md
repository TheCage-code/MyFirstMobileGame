# My First Mobile Game

A 2D mobile platformer game built with Unity. This is my second Unity project and my first mobile game.

## 🎮 About
Unlike my first project, I developed most of this game independently without following tutorials. I paused and challenged myself to implement many systems on my own — such as character movement, enemy animations, and Unity configurations. Since it was my first mobile game, I referenced tutorials mainly for the mobile-specific parts like AdMob integration.

Throughout development I encountered many bugs, and I was able to debug and solve most of them on my own.

## 🛠️ What I Learned & Implemented

**Player & Enemy**
- Character movement system (coded independently)
- Enemy animations and logic (coded independently)

**UI & Canvas**
- Level selection screen
- In-game UI elements

**Mobile**
- AdMob banner ad integration
- Mobile input handling
